DOWN-WITH-THE-FITNESS

Down-With-The-Fitness is an application created to get you moving! All users are able to view standard workouts with different focuses on specific muscle groups and our signed-in users are able to create and save custom created workouts. All users are able to start a workout and go through an exercise circuit with a running timer.

Github Link: https://github.com/laperzyna/down-with-the-fitness
