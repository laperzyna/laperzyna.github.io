THE FEED

Welcome to THE FEED! This project was completed as part of a Visual Communication 2 course. The premise of this project is to collect “your feed,” or online activity, and create a social media type application that shows “your feed.“ I designed this application is a way that shows the computer as a person that is constantly downloading your feed. It is meant to invoke a sense of slight fear and a feeling of being overwhelmed by all of your collected data.

Github Link: https://laperzyna.github.io/thefeed/
