ARCHITECTURAL ABYSS

This piece was created as part of a project for a Design Fundamentals class. The goal of this project was to become familiar with Adobe Illustrator and create a successful reflected design using one square. The square is formed from different photos of architecture and reflected multiple times to create a pattern. The entire piece was also only allowed three hues.
